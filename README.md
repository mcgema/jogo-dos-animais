## Rodando o Projeto

1. Abra o terminal da sua escolha;
2. Execute ```swipl``` para ativar o [SWI-Prolog](https://www.swi-prolog.org/);
3. Execute o jogo, com o comando a seguir:

```prolog
?- consult("main.pl").
?- play.
```
